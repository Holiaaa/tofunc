# tofunc (0.1.6)

A small, extensible Python package for translating coefs made by scipy and numpy (and other) into readable functions (latex support!)

## Features
- Transform coefs generated by scipy, numpy into readble function (latex support!)
- Polynomial, exponential, log, linear...

## Installation
Install it by doing :
```bash
git clone https://github.com/Holiaaa/tofunc.git
cd tofunc
pip install -e .
```

## Quickstart
Simple exemple for polynomial function :
```py
from tofunc import to_poly2, to_latex
from scipy.optimize import curve_fit
import numpy as np

x = np.arange(10)
y = 2*x**2 + 3*x + 1 + np.random.randn(10)*5 # for noise

def poly2(x, a, b, c):
    return a*x**2 + b*x + c

params, _ = curve_fit(poly2, x, y)
a, b, c = params

print(to_latex(to_poly2(params)))
```

CLI:
It will output something like :
```bash
3.O2424982x^{2} + 7.9289902x + 8.29429
```

## API Reference
```py
to_poly2(params) # Return a degree 2 polynomial function.
to_poly3(params) # Return a degree 3 polynomial function.
to_polyn(n, params) # Return a degree n polynomial function.
to_log(params) # Return a log function.
to_exp(params) # Return a exponential function.
to_linear(params) # Return a linear function.
to_latex(params) # Return a latex version of the function.
```
